## Codebase Patterns

- Effect-TS services use `Effect.Service<T>()('@app/Name', { effect: ... })` with `static layer = this.Default; static Live = this.layer.pipe(Layer.provide(ConfigLive))`
- Config via `Config.string()` / `Config.redacted()`, exception: `process.env.TELEGRAM_WEBHOOK_SECRET` in route
- Domain queries in `lib/core/<domain>/queries.ts`, return `Effect.Effect<T, E, Db>`
- Errors as `Data.TaggedError` in `lib/core/errors/` and `lib/services/*/errors.ts`
- All files kebab-case, imports via `@/` alias, no barrel files
